# Score Explanation Wireframe

Page section: "Score Breakdown"

Header row:
- Left: "Score Breakdown" title
- Right: Final Score badge (e.g., 72/100)

Grid of 4 cards (2 columns on desktop, 1 column on mobile):

Each card shows:
- Component title (ATS Readability, Skill Match, Experience Relevance, Resume Structure)
- Weight (percentage)
- Component score (0-100)
- Short explanation line ("Why this score?")
- Two stacked lists:
  - "Helped" (green bullets with 1-2 items)
  - "Dragged Down" (amber bullets with 1-2 items)

Visual cues:
- Green list indicates positives that lifted the score.
- Amber list indicates issues that lowered the score.
