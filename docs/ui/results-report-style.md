# Results Report Style Contract

Use this as a quick reference to keep Results report styling consistent.

- Base font size: 16px
- Minimum meta text size: 14px
- Line height: 1.65–1.75
- Text color tiers:
  - Primary: gray-900
  - Secondary: gray-700
  - Meta: gray-500
- Card variants: `base`, `emphasis`, `muted`
- Primary blue: `blue-600`
