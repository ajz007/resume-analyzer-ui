# Skill Gap Analyzer - UI Layout

Section: "Skill Gap Analysis"

Subtitle: "Compared to the job description."

Layout:
- 3 cards in a grid (1 column on mobile, 3 on desktop)
  - Card 1: Strong match (green)
  - Card 2: Partial match (amber)
  - Card 3: Missing but important (rose)

Each card lists skills with:
- Skill name (bold)
- Reason line
- Recommendation line (for partial/missing)

Bottom panel:
- "Recommendations" list with 2-3 short tips on what to do next.
